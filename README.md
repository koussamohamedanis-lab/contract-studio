# Contract Studio

Tool for building and editing personalized contracts with live preview.

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm start
```

3. Build for production:

```bash
npm run build
```

## About

This React app helps you create and customize contract templates and preview them live. Export or print contracts for signing.
